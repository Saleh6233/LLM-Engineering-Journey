# 🧠 LLM Engineering Journey

Welcome to my LLM Engineering repository.

This repository documents my hands-on journey through the theory, implementation, training, analysis, alignment, and fine-tuning of Large Language Models (LLMs). The goal is to build a deep understanding of modern LLM systems by implementing core concepts from scratch and exploring advanced techniques used in production-grade models.

All implementations are built using Python and PyTorch, with a strong focus on understanding the underlying mathematics, architecture, and engineering trade-offs.
